# cveraptor

A Python CVE intelligence CLI that enriches vulnerabilities with CVSS, EPSS, KEV, and exploit context.

## Planned Features

- CVE lookup by ID
- CVSS v3.1 and v4.0 parsing
- EPSS score and percentile
- CISA KEV status
- Exploit and PoC context
- Risk-prioritised terminal output

## Example

```bash
cveraptor CVE-2026-21571# CVEraptor
